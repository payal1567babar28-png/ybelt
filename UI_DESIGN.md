# StarVote UI Design

## Color Palette
- **Ink** (#0d0d0d) - Primary dark color
- **Paper** (#f0ede6) - Primary light/background
- **Teal** (#0e7c7b) - Accent (success, positive)
- **Amber** (#e67e22) - Secondary accent (pending)
- **Red** (#c0392b) - Error/alert

## Typography
- **Headers**: Playfair Display (serif, elegant)
- **Body**: Space Grotesk (modern sans-serif)
- **Code**: JetBrains Mono (monospace)

## Components
- Ticker bar with live poll updates
- Navigation with pill buttons
- Hero section with voting options
- 2-column info panels
- Live event feed
- Transaction tracking display

## Animations
- Ticker scroll animation
- Pulse effects on active buttons
- Glow effects on results
- Smooth fill transitions
- Status indicator pulses
